Adds \"email confirmed\" to the PMPro members list CSV.

== Description ==
Adds an \"email confirmed\" boolean column to the Paid Membership Pro members list CSV.  It is true after a new user has followed the confirmation link in the registration email.

For use with PMPro\'s Email Confirmation Add On (http://www.paidmembershipspro.com/add-ons/plus-add-ons/email-confirmation-add-on/).

== Installation ==
Upload to wp-content/plugins and activate the plugin.
